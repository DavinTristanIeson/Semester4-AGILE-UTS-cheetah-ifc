- POST /api/accounts/register:
  Endpoint ini digunakan untuk membuat akun baru dalam sistem.


  Request:

  Headers:
  Content-Type: multipart/form-data
  Body (FormData):
  email (string, required): Alamat email yang valid untuk akun baru.
  password (string, required): Kata sandi untuk akun baru.
  nama (string, required): Nama lengkap pengguna.
  deskripsi (string, optional): Deskripsi singkat tentang pengguna.
  pfp (file, optional): File gambar profil pengguna (Profil Picture).


  Response:

  Status Code 201 (Created):

  Body:
  id (string): ID unik yang diberikan kepada akun yang baru dibuat.
  email (string): Alamat email pengguna.
  nama (string): Nama lengkap pengguna.
  deskripsi (string): Deskripsi singkat tentang pengguna.
  pfp_url (string): URL tempat penyimpanan gambar profil pengguna.
  Deskripsi: Menunjukkan bahwa akun baru berhasil dibuat dan data pengguna telah dimasukkan ke dalam database.

  Status Code 400 (Bad Request):

  Body:
  message (string): Pesan yang menjelaskan kesalahan atau masalah yang terjadi.
  Deskripsi: Menunjukkan bahwa format data yang dikirim dalam permintaan tidak sesuai dengan yang diharapkan. Pesan kesalahan harus     menjelaskan masalah yang spesifik, seperti format yang salah atau kolom yang hilang.

  Status Code 409 (Conflict):

  Body:
  message (string): Pesan yang menjelaskan bahwa email yang digunakan sudah terdaftar.
  Deskripsi: Menunjukkan bahwa alamat email yang digunakan untuk membuat akun baru sudah terdaftar dalam sistem. Pengguna harus menggunakan    email yang unik.


- POST /api/accounts/login:
  Endpoint ini digunakan untuk melakukan login dan melakukan otentikasi dengan menggunakan cookie.

  Request:

  Headers:
  Content-Type: application/json
  Body:
  email (string, required): Alamat email yang terdaftar.
  password (string, required): Kata sandi yang sesuai dengan akun pengguna.

  Response:

  Status Code 200 (OK):

  Set-Cookie:
    name: session
    value: <session_token>
    path: /
    HttpOnly
  Deskripsi: Menunjukkan bahwa otentikasi berhasil dan mengatur cookie sesi untuk pengguna.
  Status Code 401 (Unauthorized):

  Body:
  message (string): Pesan yang menjelaskan bahwa otentikasi gagal.
  Deskripsi: Menunjukkan bahwa otentikasi gagal karena kredensial pengguna tidak valid atau tidak ditemukan.
  Status Code 400 (Bad Request):

  Body:
  message (string): Pesan yang menjelaskan kesalahan atau masalah yang terjadi.
  Deskripsi: Menunjukkan bahwa format data yang dikirim dalam permintaan tidak sesuai dengan yang diharapkan. Pesan kesalahan harus     menjelaskan masalah yang spesifik, seperti format yang salah atau kolom yang hilang.


- GET /api/accounts/me:
  Endpoint ini digunakan mengecek apakah pengguna terotentikasi.

  Request:
  Method: GET
  Path: /api/accounts/me
  Headers:
    Cookie: session=<session_token>

  Response:

  Status Code 200 (OK):
  Deskripsi: Menunjukkan bahwa pengguna terotentikasi.

- GET /api/accounts/:id:
  Endpoint ini digunakan untuk mengambil informasi akun pengguna berdasarkan ID.

  Request:

  Method: GET
  Path: /api/accounts/:id
  Response:

  Status Code 200 (OK):

  Body:
  id (string): ID unik akun pengguna.
  email (string): Alamat email pengguna.
  nama (string): Nama lengkap pengguna.
  deskripsi (string): Deskripsi singkat tentang pengguna.
  pfp_url (string): URL tempat penyimpanan gambar profil pengguna.
  Deskripsi: Menunjukkan bahwa informasi akun pengguna berhasil ditemukan dan dikembalikan.
  Status Code 404 (Not Found):

  Deskripsi: Menunjukkan bahwa tidak ada akun pengguna yang ditemukan berdasarkan ID yang diberikan.
